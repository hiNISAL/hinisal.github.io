闭包与作用域

闭包在很多语言中都存在 比如python、lua等

不了解闭包 但是不一定没用到过

function func(){var a = 1; return a;}
func();

// 1
通常情况下 这样的一个函数声明的函数 在调用之后 内部的变量等都因为垃圾回收机制被回收了 不会保留在内存中

function func(){var a = 1; return function(){return a;}}
var func2 = func();
func2();

// 1
但是在这种情况下 我们把一个函数当作返回值返回 并且这个函数的作用是返回上层函数的a 这样我们调用这个函数后 a这个变量会一直留在内存中 不会被释放

这种情况就是闭包

闭包十分得常见 比如在使用jquery的ajax的时候

(function(){
  var msg = “Success.”;
  var url = “./test.php”;
  $.ajax({
    url: url,
    success: function(){
      alert(msg);
    }
  });
})();

在得到服务端返回的http请求之后 会调用success这个函数 这个函数可以访问到他上层函数或者说外部的变量 这里就是因为闭包

对象的get/set方法

js在申明对象的时候 定义set和get方法

那具体有什么用呢

比如说 我们在修改一个对象的属性值的时候 我们想出发一系列的别的操作 但是单纯的通过对象.属性 = 属性值这种方式 除了赋值以外 一般情况下不会再做别的操作 这个时候就可以用get、set方法

var person = {
  name: ‘nisal’,
  age: 16,
  _weight: 0,
  get weight(){
    alert(this._weight);
  },
  set weight(val){
    val < 100 ? this._weight = val : alert(“太*了”);
  }
};
person.weight;
person.weight = 1000;
person.weight;

// 0
// 太重了
// 10
