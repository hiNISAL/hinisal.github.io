LESS笔记

1. 注释
   //   这种注释不会被编译
   /**/ 会被编译

  1. 变量
    使用 @ 符号定义一个变量
    @red: #f00;

  2. mixin
    1) 比如:
    .box{

    background: #f00;
.border;

    }
    .border{

    color: #ff0;

    }
    编译后:
    .box{

    background: #f00;
color: #ff0;

    }
    .border{

    color: #ff0;

    }

    2) 比如 带参数的:
    .box{

    background: #f00;
.border(10px);

    }
    .border(@borderWidth){

    border: solid @borderWidth #f00;

    }
    编译后:
    .box{

    background: #f00;
border: solid 10px #ff0000;

    }
    3) 比如 带参数并且有默认值的:
    .box{

    background: #f00;
.border;

    }
    .border(@borderWidth: 50px){

    border: solid @borderWidth #f00;

    }
    编译后:
    .box{

    background: #f00;
border: solid 50px #ff0000;

    }

  3. 匹配模式【类似函数重载】
    比如:
    .border(left){border-left: 1px solid #f00;}
    .border(right){border-right: 1px solid #f00;}
    .box1{.border(left);}
    .box2{.border(right);}
    编译后:
    .box1 {border-left: 1px solid #f00;}
    .box2 {border-right: 1px solid #f00;}
    同时 匹配的时候可以选择默认肯定加载的一个样式
    .border(left){border-left: 1px solid #f00;}
    .border(right){border-right: 1px solid #f00;}
    .border(@_){color: #f00;}
    .box1{.border(left);}
    .box2{.border(right);}
    编译后
    .box1 {
    border-left: 1px solid #f00;
    color: #f00;
    }
    .box2 {
    border-right: 1px solid #f00;
    color: #f00;
    }

  4. 运算
    数字 颜色 变量都能参与运算
    比如:
    @w300: 300px;
    .box{
    width：@w300 + 20;
    color: #ccc - 10;
    }

  5. 嵌套规则
    1) 原生:
    li{

    // ...

    }
    li a{

    // ...

    }
    li a:hover{

    // ...

    }

    less中:
    li{

    // ...
a{
  // ...
  &amp;:hover{
    // ...
  }
}

    }

  6. @arguments变量 和js中的arguments意义相同
    比如
    .border(@w: 30px, @c: #ff0, @l: solid){
    border: @arguments;
    }
    .box{
    .border;
    }
    编译后:
    .box {
    border: 30px #ffff00 solid;
    }

  7. 避免编译
    比如:
    .box{
    width: calc(100% - 30px);
    }
    编译后
    .box {
    width: calc(70%);
    }

    less会帮你计算 但是这部分应该是浏览器计算
    为了避免这种问题 可以用 ~’’ 把不需要编译的内容包起来 避免编译
    比如
    .box{
    width: ~’calc(100% - 30px)’;
    }
    编译后
    .box {
    width: calc(100% - 30px);
    }

  8. !important 关键字
    如果用在mixin之后 那么所有的样式都会被加上
    比如:
    .boxBase{
    width: 100px;
    height: 100px;
    }
    .box{
    .boxBase !important;
    }
    编译后:
    .boxBase {
    width: 100px;
    height: 100px;
    }
    .box {
    width: 100px !important;
    height: 100px !important;
    }
