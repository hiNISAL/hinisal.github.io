SASS笔记


当一个sass或者scss文件为_开头的时候 不会被编译

  1. 变量
    使用 $ 开头
    比如:
    $red: #f00;
    使用:
    p{color: $red;}

  2. import
    使用 @import 来引入文件
    比如:
    @import “./var.scss”

    sass中的import和原生有区别
    sass中如果要使用原声的import的时候

    1. 引用的文件是 .css
    2. 引用的事非本地资源的时候 比如http:// 开头
    3. 使用url()的时候
    4. 带有 media queries的时候

    另外可以同时import多个文件 之间用逗号分隔
    比如:
    @import “a”, “b”;

  3. 注释
    sass可以使用 // 来进行行注释 但是编译后将会被去掉

  4. 父子元素关系
    在sass中可以直接嵌套进去
    比如:
    p{
    font-size: 10px;
    }
    p span{
    font-size: 20px;
    }

    可以写成
    p{
    font-size: 10px;

    span{

    font-size: 20px;

    }
    }

    同理 可以使用伪类伪元素
    但是必须注意不能写错
    错误写法:
    a{
    :hover{

    color: #f00;

    }
    }
    这样写诗错误的 编译后的结果是:
    a *:hover{color: #f00;}

    sass提供了一个方法来引用当前类:
    a{
    &:hover{

    color: #f00;

    }
    }
    使用&符号来引用当前的a

  5. 属性嵌套
    比如 font相关的设置 在原生css中可以简写 但是不会太直观
    在sass中 提供了简写属性的嵌套写法:
    p{
    font: {

    family: Microsoft Yahei;
size: 10px;

    }
    }
    这个等同于写:
    p{
    font-family: Microsoft Yahei;
    font-size: 10px;
    }
    和
    p{
    font: Microsoft Yahei, 10px;
    }

  6. 函数操作
    1)hsl函数在原声css中存在兼容性问题
    在sass中 hsl函数在经过编译后会转成16进制颜色

可以用 @Function 来申明函数

使用@mixin 定义一个样式 可以传入参数 比如 @mixin demo{font-size: 10px} 用法 p{@include demo();}
含有参数的mixin @mixin fs ($fontsize){font-size: $fontsize} 用法 p{@include fs(10)}
含有参数和默认值的mixin @mixin fs ($fontsize: 10){font-size: $fontsize} 用法 p{@include fs()}

  1. 继承
    可以通过@extend 继承另一个样式的属性
    比如:
    .worning{color: #f00;}
    .error{@extend .worning; text-weight: 900;}

    编译后:
    .worning, .error{color: #f00;}
    .error{text-weight: 900;}

    支持继承多个选择器
    但是

    1. 不能继承选择器序列 比如 .a .b{color: #f00;}
    2. 使用百分号 可以设置这个样式只能用来继承 而不会变编译 比如: %worning{color: #f00} .error{@extend %worning; font-weight: 900;}

7.sass中的@media querires
sass中的mediaqueiry可以内嵌在css规则中 生成css的时候 才会被提到样式最高层级
好处： 避免了重复书写选择器或者打乱样式表流程

比如:
@mixin col-sm($width: 50%){
$media(min-width: $width; float: left;)
}

7.@at-root指令
在嵌套的时候 可以用@at-root指令 将嵌套的元素 拉到顶端
比如:
p{
@at-root{
span{
color: #f00;
}
}
span{
color: #ff0;
}
}
编译后:
span{color: #f00;}
p span{color: #ff0;}

8.条件判断
sass有if指令进行条件判断
sass有type-of来判断参数的类型
比如:
@mixin col-sm($width: 50%){
@if type-of($width) != number{
@error “比如是一个字符串 你输入的事 #{$width}”;
}
@media(min-width: 768px){
width: $width;
float: left;
}
}
其中 在字符串中引用变量 要用 #{$varname} 这样的格式

另外还能有与或非操作
比如:
@mixin col-sm($width: 50%){
@if type-of($width) != number{
@error “比如是一个字符串 你输入的事 #{$width}”;
}
@if not unitless($width){ // 判断是不是有单位
@if unit($width) != “%” { // 判断单位是不是百分号
@error “必须传数且是百分值”
}
} @else {
@warn “警告 要输入一个百分值”; // 发起一个警告
$width: (precentage($width)) ／ 100; // 把变量添加百分比单位
}
@media(min-width: 768px){
width: $width;
float: left;
}
}

9.输出模式

output_style = :expanded or :nested or :compact or :compressed

  1. expanded // 最常见的样式 也是默认样式
  2. nested // 对应html结构进行缩紧
  3. compact // 体现选择器之间的关系
  4. compressed // 压缩
